Constructor takes in object id, path to urdf or sdf files (If one exists) and an object name. This serves as the base class that all other object types should inherit from.

Parameters

  • id (int) – The id returned from pybullet when an object is loaded in.

  • path (str) – Path to the urdf or sdf file that was used.

  • name (str) – Name of the object.
