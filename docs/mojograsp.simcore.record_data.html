RecordDataJSON Class (Optional default class for recording data to json files)

__init__(data_path: ~typing.Optional[str] = None, data_prefix: str = 'episode', save_all=False, save_episode=True, state: ~mojograsp.simcore.state.State = <mojograsp.simcore.state.StateDefault object>, action: ~mojograsp.simcore.action.Action = <mojograsp.simcore.action.ActionDefault object>, reward: ~mojograsp.simcore.reward.Reward = <mojograsp.simcore.reward.RewardDefault object>)

Constructor takes in State, Action and Reward objects from the user and saves their outputs at every timestep. Optionally can save each episode individually, alltogether, or not at all. State, Reward and Action defaults are given if user does not define them (they will return empty dictionaries).

Parameters

  • data_path (str) – Desired directory path for storing resulting json files.

  • data_prefix (str) – Named prefix before each json file. Defaults to episode

  • save_all (bool) – Set whether or not you would like to save all episodes into one file (Default False).

  • save_episode (bool) – Set whether or not you would like to save each episodes into individual files (Default True).

  • state (State()) – State object defined by user, if none given then StateDefault() will be used.

  • action (Action()) – Action object defined by user, if none given then ActionDefault() will be used.

  • reward (Reward()) – Reward object defined by user, if none given then RewardDefault() will be used.

record_timestep()

Method called by SimManager() every timestep. Records the user defined state, action, and reward. Saves the result to a timestep dictionary which is later used to form an episode.

record_episode()

Method called by SimManager() after every episode. Compiles all of the timestep dictionaries into a list and adds it to the episode dicionary.

save_episode()

Method called by SimManager() after every episode. Saves the most recent episode dictionary to a json file.

save_all()

Method called by SimManager() after all episodes are completed. Saves all episode dictionaries to a json file.
