mojograsp.simcore.action module

class Action

Bases: ABC

Action Abstract Base Class

abstract __init__()

Constructor should be used to pass any other objects or data that an action will need to function. Usually this will be a manipulator or other mojograsp objects.

abstract set_action() → dict

Method should be used to set the class variable self.current_action. It is called by the sim manager BEFORE the simulator is stepped. This can be set however you would like, as long as self.current_action is updated so that get_action is returns properly. This could include joint target angles, end effector targets, etc.

abstract get_action() → dict

Method should return a dictionary that represents the intended target action BEFORE it is executed and the sim is stepped. The return value is class variable self.current_action which is updated by set_action().

Returns

Dictionary containing the representation of an intended action for a simulator object.

Return type

Dictionary with format {string: ANY TYPE}.
